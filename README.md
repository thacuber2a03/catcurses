# catcurses

ncurses, abstracted around [Umka](https://github.com/vtereshkov/umka-lang).

```umka
import (
	cc = "../catcurses.um"
)

fn main() {
	t := cc::stdTerminal()
	t.raw(true)
	t.echo(false)
	t.halfDelay(1)

	w := t.window
	w.keypad(true)

	x, y := 1, 2

	for true {
		c, ok := w.getKey()
		if c == cc::Key('q') { break }

		switch c {
		case .left: x--
		case .right: x++
		case .up: y--
		case .down: y++
		}

		w.clear()
		w.printAt(0, 0, "(Use the arrow keys to move the text around)")
		w.printAt(x, y, "Hello from catcurses!")
		w.refresh()
	}

	t.destroy()
}

```
